# Go File Handling Packages

| Package    | When to Use                                  | Key Idea                           |
| ---------- | -------------------------------------------- | ---------------------------------- |
| os         | Basic file operations (open, create, delete) | Direct interaction with OS         |
| io         | Moving data between sources and destinations | Abstraction via Reader/Writer      |
| bufio      | Large files or line-by-line processing       | Buffered I/O for performance       |
| fmt        | Writing or reading formatted text            | Human-readable structured output   |
| filepath   | Handling file paths across OS                | Safe and portable path handling    |
